Model

おはじきを使って掛け算の意味（「1つ分の数」×「いくつ分」＝「全部の数」）を視覚的に理解できるウェブアプリを作成しました。

数字という「記号」と、おはじきやお皿という「実体」を頭の中で結びつける（記号接地）ために、色分けを工夫しています。

  • ピンク色: 1つ分の数（おはじきの数）

  • 水色: いくつ分（お皿の枚数）

このように色を統一することで、数式の数字が現実のどの部分を指しているのかが直感的にわかるように設計しています。

おすすめの遊び方・ポイント

  1. 色のリンクを意識する: 問題文の「ピンクの文字（○個）」とお皿の中の「ピンクのおはじき」、問題文の「水色の文字（○枚）」と「水色のお皿の枠」がリンクしています。お子様と一緒に「1つのお皿に何個入っているかな？（ピンク）」「お皿は全部で何枚あるかな？（水色）」と指差しながら確認するとより効果的です。

  2. 全部でいくつ？: 答えを入力する前に、実際におはじきのイラストを1つずつ数え上げることで「掛け算は足し算の繰り返しである」という概念（全部の数）を体感できます。

  3. スマホ・タブレット対応: 画面サイズに合わせてレイアウトやおはじきのサイズが変わるため、タブレットなどでお子様が直接タッチして数えやすいように設計してあります。

Model

かけ算の導入として最も素朴でわかりやすい「縦横の配列（アレイ図）」を使った**「きほん編」を追加し、前回のお皿とおはじきを「はってん編」**として選べるようにバージョンアップしました。

きほん編では、**「1列に並んだ数（ピンク）」と「それが何段あるか（水色）」**を視覚的に結びつけるため、各段に水色の番号札をつける工夫をしています。

アップデートのポイント

画面上部に**「🌱きほん（たてよこ）」と「🚀はってん（おさら）」**の切り替えタブを追加しました。

【きほん（たてよこ）モードの工夫】

かけ算の根本である「アレイ図（ものを縦横に規則正しく並べた図）」を採用しています。

  1. 「1つ分の数」の強調: 横1列に並んだおはじきを、うっすらとピンクの枠で囲むことで「これが1まとまり（A個）だよ」ということを視覚的に表しています。

  2. 「いくつ分」の強調: それぞれの段の左側に「1、2、3...」と水色の番号札をつけています。これにより「このまとまりが全部で何段（B段）あるか」がはっきりと数えられるようになります。

まずは「きほん」で『1列に○個、それが○段』という規則的な数の増え方（アレイ図）を理解し、慣れてきたら「はってん」でお皿などのグループ分け（包含除の基礎）へとステップアップして遊んでみてください。

Model

言葉や装飾をそぎ落とし、ただ「数」と「空間（配列）」の事実に向き合うためのミニマルなアプリケーションを作成しました。

縦と横の区別をなくすため、色は単一（スレートグレー）に統一しています。また、「回転（↻）」ボタンを配置し、図形を90度回転させることで、A×B と B×A が本質的に同じ形・同じ数であること（可換性・対称性）を、言葉を一切使わずに直感的に確かめられるようにしました。

意図と設計思想

  • 完全な対称性: 図形は「点」ではなく「角丸の正方形（マス目）」で構成しています。これにより、将来的に面積モデル（長方形の面積）やヤング図形などに移行しやすい抽象度を持たせました。

  • 回転操作: 中央の「↻」ボタンを押すと配列全体が90度回転します。数式の A と B は変わりませんが、図の縦横が入れ替わることで、対称群としての操作や「交換法則」を無言で体験できます。

  • 説明の排除: 「正解！」「次へ」といった文字やボタンをすべて消しました。正しい答えを入力した瞬間にだけ色が変わり、少しの間を置いて静かに次の問題へと切り替わります。事実のみを提示するストイックなインターフェースです。
