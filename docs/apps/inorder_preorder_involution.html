Treapの Inorder / Preorder 対合 (n=3)

長さ n=3 の配列 [1,2,3] の順列からなる inorder と preorder のペアのうち、有効な二分木を構成できる全30通りを列挙しています。
それぞれの木を Treap（デカルト木） と見立て、各ノードの論理座標を (x, y) = (inorderのインデックス, preorderのインデックス) として平面上に配置しました。

配列の並びを逆順にし、inorderとpreorderを入れ替えるという操作：
New Inorder = reverse(Preorder)
New Preorder = reverse(Inorder)
を行うと、得られた新しいペアも必ず有効な二分木を構成します（この変換は2回行うと元に戻る対合です）。

幾何学的には、この変換は平面上で 直線 y = -x + 2 （赤い破線）に関する折り返し対称移動 に相当し、各ノードの座標が (x, y) → (2-y, 2-x) に移る様子がはっきりと観察できます。
